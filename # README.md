# README.md

A modern, premium **personal portfolio landing page** built using **vanilla HTML, CSS, and JavaScript** with a dark futuristic theme, neon highlights, glassmorphism effects, and smooth animations. The page showcases your profile, experience, featured projects, skills marquee, social links, and resume download option.

## üöÄ Features

* Futuristic glassmorphic UI
* Responsive layout (mobile-friendly)
* Floating animations & scroll-reveals
* Skills marquee (infinite scroll)
* Featured projects with clickable launch buttons
* Social profile integration
* Ready for deployment via **Vite**, GitHub Pages, Netlify, or Vercel

## üìÇ Project Structure

```
index.html         # Main landing page
logo.jpg           # Brand/portfolio logo
portfolio image.jpg # Hero/profile image
about.png          # About section image
Audio_to_text.png  # Project thumbnail
Karthikeyan_S__Resume.pdf  # Resume file
```

## üîß Usage

1. Place all assets (images, resume PDF) in the same folder as `index.html`.
2. If using **Vite**, move assets into `/public` or import them correctly.
3. Open the file in a browser or deploy to any static hosting.

## üåê Deployment

This project works seamlessly on:

* GitHub Pages
* Vercel
* Netlify
* Cloudflare Pages
* Any static site server
